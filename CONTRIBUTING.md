# Contributing

Contributions must preserve the no-raw-export doctrine.

## Required

- No raw titles in exported proof bundles
- No raw URLs in exported proof bundles
- No server/hostname preservation in exported proof bundles
- No media files in proof packs
- No user data
- Leak scanner must pass
- Replay must be deterministic for fixed input/seed/epoch

## Good contributions

- Adapter profiles
- Category maps
- CLI validation improvements
- ARC-Core integration
- Arc-RAR packaging improvements
- OmniBinary binary format improvements
- Ed25519 signing for v1.0
