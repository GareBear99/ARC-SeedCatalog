# Legal Policy

ARC-SeedCatalog is technically source-agnostic but supported only for lawful ingestion.

Allowed:

- public-domain catalogs
- licensed APIs
- owned/internal media libraries
- homebrew/demo catalogs
- synthetic test feeds

Disallowed:

- piracy-site mirroring
- unlicensed streaming link routing
- hidden host/server preservation
- raw stream URL storage
- access-control bypass
- user tracking without consent
