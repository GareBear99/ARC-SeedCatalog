# Arc-RAR Export

The `arc_rar_export_bundle` is a manifest for portable proof packaging.

It includes:

- receipt bundle hash
- policy bundle hash
- index bundle hash
- ARC-Core bundle hash
- OmniBinary bundle hash

It excludes:

- raw input JSON
- resolver maps
- titles
- paths
- URLs
- server names
- hostnames
- media
- user data
