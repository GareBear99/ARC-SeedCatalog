ARC-SeedCatalog v0.4 · complete static split-bundle release

Ingest authorized source JSON. Export ARC-Core, Arc-RAR, and OmniBinary proof bundles.

A no-backend GitHub Pages tool that converts flexible server/source/catalog JSON into opaque seeded receipts, category vectors, ARC-Core registration JSONL, Arc-RAR manifests, and OmniBinary hash reports.

For lawful/authorized/public-domain/licensed/internal catalogs only. Exported bundles do not store raw server names, titles, paths, URLs, hostnames, media, descriptions, posters, or user data.
